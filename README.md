# Tamil Nadu Election Analysis 2026

## Overview
This project analyzes the 2026 Tamil Nadu Assembly Election using publicly available Election Commission of India (ECI) constituency-level data.

The objective was to identify the most important electoral shifts and present them as a newsroom-style data storytelling deck for AtliQ Media.

---

## Key Insights

- TVK emerged as a major competitive force across Tamil Nadu.
- Chennai Metro showed the strongest political realignment.
- Urban constituencies demonstrated the sharpest vote share shifts.
- Higher turnout aligned closely with stronger TVK momentum.

---

## Analysis Workflow

The analysis was primarily performed using Microsoft Excel.

Tasks included:
- regional aggregation
- vote share analysis
- turnout trend identification
- chart preparation
- storytelling presentation development
