# Tamil Nadu Election Analysis 2026

## Overview
This project analyzes the 2026 Tamil Nadu Assembly Election using publicly available Election Commission of India (ECI) constituency-level data.

The objective was to identify the most important electoral shifts and present them as a newsroom-style data storytelling deck for AtliQ Media.

---

## Key Insights

- TVK emerged as a major competitive force across Tamil Nadu.
- Chennai Metro showed the strongest political realignment.
- Urban constituencies demonstrated the sharpest vote share shifts.
- Higher turnout aligned closely with stronger TVK momentum.

---

## Analysis Workflow

Most analytical preprocessing and aggregation were performed in Microsoft Excel before visualization and presentation.

Tasks included:
- regional aggregation
- vote share analysis
- turnout trend identification
- chart preparation
- storytelling presentation development

## Python Notebook

A basic Python notebook (`Tamil_Nadu_Election_Analysis.ipynb`) is included to demonstrate simple reproducible analysis using pandas and matplotlib.

The notebook performs:
- Data loading using pandas
- Filtering TVK vote data
- Regional vote aggregation
- Basic visualization using matplotlib

---

## Reproduction Steps

1. Download the repository files.
2. Open the Excel dataset (`Tn_Election_2026.xlsx`).
3. Open the notebook file (`Tamil_Nadu_Election_Analysis.ipynb`) using Google Colab.
4. Run all notebook cells sequentially.
5. The notebook reproduces the regional TVK vote aggregation and visualization used for the analysis.
